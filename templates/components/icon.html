Icon

Displays an icon from the Phosphor Icons library.

Example

Usage

<go-icon name="heart" weight="fill" size="lg"></go-icon>

Sizes & Weights

sm

md

lg

xl

API

Attribute Type Default Description
name string question Icon name (e.g., 'heart', 'user', 'arrow-right')
size string md Size: 'sm', 'md', 'lg', 'xl', 'xxl' or CSS value
weight string regular Weight: 'thin', 'light', 'regular', 'bold', 'fill', 'duotone'

See all available icons at phosphoricons.com.
